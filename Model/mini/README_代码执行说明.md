# 上海口袋公园花坛美化效果评价与多目标优化配置研究

## 代码文件结构

本项目包含6个核心Python脚本，按以下顺序执行：

### 执行顺序

| 序号 | 文件名 | 功能描述 |
|:----:|--------|----------|
| 0 | `00_read_data.py` | 数据读取模块 - 读取原始数据并生成缓存 |
| 1 | `01_create_templates.py` | 创建数据模板 - 生成7个基础数据模板文件 |
| 2 | `02_plant_database.py` | 植物数据库构建 - 处理4个附件生成完整植物数据库 |
| 3 | `03_data_analysis.py` | 小红书数据分析 - 关键词提取、情感分析、词频统计 |
| 4 | `04_modeling.py` | 建模分析 - 熵权法、TOPSIS评价、K-Means聚类 |
| 5 | `05_visualization.py` | 综合可视化 - 生成所有分析图表 |

## 执行方法

### 方法一：按顺序手动执行

```bash
# 1. 数据读取
python 00_read_data.py

# 2. 创建数据模板
python 01_create_templates.py

# 3. 构建植物数据库（如有附件数据）
python 02_plant_database.py

# 4. 小红书数据分析
python 03_data_analysis.py

# 5. 建模分析
python 04_modeling.py

# 6. 生成可视化图表
python 05_visualization.py
```

### 方法二：批量执行

```bash
# 在命令行中依次执行
for f in 0*.py; do python "$f"; done
```

## 各模块功能详解

### 第0章：数据读取 (`00_read_data.py`)

**功能**：
- 读取上海口袋公园CSV数据
- 读取并合并多个小红书评论CSV文件
- 支持多种编码格式自动识别（UTF-8、GBK、GB2312等）
- 智能去重处理
- 生成pickle缓存文件

**输入**：
- `上海数据_全量_20260415104223.csv`
- `xiaohongshu*.csv` 系列文件

**输出**：
- `df_shanghai.pkl_2`
- `df_xiaohongshu.pkl_2`

---

### 第1章：数据模板创建 (`01_create_templates.py`)

**功能**：
- 创建区级绿化指标数据
- 创建口袋公园名录数据
- 创建乡土植物特性数据库
- 创建成本参考数据
- 创建公众满意度调查问卷模板
- 创建数据标准化处理模板
- 创建熵权法计算模板
- 创建花卉经济成本数据库

**输出**：7个Excel模板文件 + 1个花卉成本数据库

---

### 第2章：植物数据库构建 (`02_plant_database.py`)

**功能**：
- 读取并整合4个附件数据
- 解析光照条件
- 添加花期信息（150+种植物）
- 估算种植成本
- 确定推荐场景（商圈型/社区型/干道型/街角型）
- 评估生态适应性和美学评分

**输入**：
- 附件1-4 Excel文件

**输出**：
- `上海市绿化植物完整数据库.xlsx`（多sheet）
- `上海市绿化植物完整数据库.csv`

---

### 第3章：小红书数据分析 (`03_data_analysis.py`)

**功能**：
- 提取花卉关键词并统计词频
- 提取区域关键词并统计分布
- 情感分析（正面/中性/负面）
- 季节分布分析
- 生成综合分析报告

**输出**：
- `10_小红书评论分析结果.xlsx`
- `11_花卉词频统计.xlsx`
- `12_公园词频统计.xlsx`
- `13_情感分布统计.xlsx`
- `14_花卉情感交叉分析.xlsx`
- `综合分析报告.txt`

---

### 第4章：建模分析 (`04_modeling.py`)

**功能**：
- 构建评价指标体系
- 熵权法计算指标权重
- TOPSIS法综合评价
- K-Means聚类分析（4类场景）
- 多目标优化配置方案
- 花卉性价比分析

**输出**：
- `15_熵权法指标权重.xlsx`
- `16_TOPSIS评价结果.xlsx`
- `17_口袋公园聚类结果.xlsx`
- `18_分场景花卉优化配置方案.xlsx`
- `19_花卉性价比推荐清单.xlsx`
- `建模分析报告.txt`

---

### 第5章：综合可视化 (`05_visualization.py`)

**功能**：
- 数据概览仪表盘
- 小红书分析专题图表
- 建模评价专题图表
- 花卉成本分析图表
- 优化方案专题图表
- 综合报告封面

**输出**：`result_analysis/charts/` 目录下的6张PNG图表

## 注意事项

1. **编码问题**：脚本会自动尝试多种编码格式读取CSV文件
2. **路径配置**：所有脚本使用相对路径，无需修改代码
3. **数据顺序**：建议按顺序执行，确保依赖关系正确
4. **Excel兼容性**：需要安装 openpyxl 库支持Excel操作
5. **图表字体**：需要系统中安装中文字体（SimHei、Microsoft YaHei等）
